「神隊友」與「準時下班」比高薪更重要？—— 揭密影響工作滿意度的關鍵因子
資料科學與社會研究 第6組
林渭倫、林建宏、陳興文、王皓
資料來源：2022 年 SRDA－112 年勞工生活及就業狀況調查
一、研究動機
近年勞工權益意識抬頭，職場霸凌等新聞頻繁出現。由於我們距離踏入職場不久，因此好奇：
究竟哪些因素會提升勞工的工作滿意度？
如果能找出影響關鍵，未來在選擇工作時也能作為參考。
二、研究方法與步驟
因變數眾多，若將所有類別變項展開會產生過多變數，因此採用三階段變數篩選流程：
第 1 步：LASSO Logistic Regression
移除效果過低或具共線性的變數。
第 2 步：一般 Logistic Regression
移除類別變項中組間沒有明顯差異的變數。
第 3 步：Likelihood Ratio Test
以模型貢獻度判斷該類別變項是否影響「對工作滿意」。
三階段篩選後得到最精簡、顯著的 logistic regression 模型，再進行最終分析。
三、主要發現
1. 工時與同事相處影響最大
從 odds ratio 可見最具影響力的變數是：
工作時間滿意度
同事相處滿意度
年齡
其中年齡看似影響很大，是因為 70 歲以上樣本極少（僅 3 位且皆表示滿意），使效果被放大。
這些因素最能左右日常工作感受，因此影響最大。
2. LASSO 中被淘汰的「加班時數」
加班時數 Q3 僅為 5，小多數人加班量不高。
在此低加班區間中，「滿意」與「不滿意」差異極小，使 加班時數不足以解釋滿意度，因此在 LASSO 中被淘汰。
3. 年齡與收入的關係
年齡越高，低薪比例越低；整體薪資有「隨年齡上升而上升」的趨勢。
4. 薪資 vs. 工作量感受
若員工 對工作量滿意：薪資變動影響不大。
若員工 對工作量不滿意：需要更高薪資才能維持滿意度。
→ 若企業無法大幅加薪（尤其落在 30K–40K 區間），
改善工作量感受是提升滿意度的關鍵。
四、特別的發現
1. 多數滿意度量表皆能反映整體滿意度
整體工作滿意度源自多個構面，因此大多量表都顯著。
唯一不顯著的是：
無障礙環境滿意度
可能因多數人沒有使用障礙設施，因此對滿意度影響不大。
2. 「遭遇工作困難」未達高度顯著
雖直觀上覺得越困難越不滿意，但有些員工喜歡挑戰性的工作，
困難可能提升成就感，因此未達 1% 顯著水準。
3. 年齡與滿意度的非線性關係
年輕族群：可能因經濟壓力留在不滿意的工作。
年長族群：可能已適應職務內容，或找到滿意的工作並長期留任。
五、結論
經三階段變數篩選與最終模型分析後，我們發現：
影響工作滿意度最關鍵的因素為：
工作時間滿意度
同事相處滿意度
年齡（但需注意極端值影響）
與大眾直覺「薪資最重要」不同，
日常可感受到的因素（工時、同事互動）對滿意度更為關鍵。
